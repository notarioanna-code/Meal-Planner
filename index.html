Meal Planner Setup

One-time setup to connect your shared meal planner. Follow the steps below, then paste your Firebase config.

1

Create a Firebase project (free)

Go to console.firebase.google.com, click "Add project," name it "Meal Planner," and click through (you can disable Google Analytics).

2

Add a web app

In your project, click the web icon </>, name it "Meal Planner," skip hosting, and click "Register app."

3

Copy the config

You'll see a code block with firebaseConfig = { ... }. Copy just the part inside the curly braces (including the braces) and paste below.

4

Enable Realtime Database

In the Firebase console sidebar, click "Build" then "Realtime Database." Click "Create Database," pick any location, and select "Start in test mode."
